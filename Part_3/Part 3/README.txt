Image Clustering
-------------------


Tech/framework used:

	
OS:	Windows 10	
IDE: 	PyCharm	
Pandas : 0.22.0

--------------------------


Language Used:	Python

--------------------------


How to compile and run?


Give the path and arguments through command prompt as follows:

python ImageClustering.py "image1.jpg" "image2.jpg" "image3.jpg" 

where image1, image2, image3 are the paths to the given images.

Note : Please provide the mentioned parameters through command line.
